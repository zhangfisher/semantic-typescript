### 版本 1.x.x

- 初始稳定版本
- 完整的 TypeScript 支持，严格类型
- 异步和同步流处理
- 内置统计分析
- 带有自动清理的事件流管理
- 全面的收集器系统

### 主要特性

- **零样板索引** - 所有元素上的自动 bigint 索引
- **函数式纯粹风格** - 完整的 TypeScript 类型推断
- **防泄漏事件流** - 内置的 DOM 和 WebSocket 事件安全性
- **内置统计** - 数字和 bigint 分析
- **内存高效** - 流的惰性求值
